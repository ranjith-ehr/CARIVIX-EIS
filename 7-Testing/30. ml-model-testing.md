# ML Model Testing

This document covers input validation tests, output range verification, anomaly detection accuracy benchmarks, model consistency tests, regression test triggers, and how to interpret failures.

---

## Test File Locations

```
ml/
└── tests/
    ├── test_xgboost_model.py          # Performance predictor tests
    ├── test_logistic_classifier.py    # Employee classifier tests
    ├── test_isolation_forest.py       # Anomaly detector tests
    ├── test_prophet_forecast.py       # Attendance forecaster tests
    ├── test_shap_explainer.py         # SHAP value generation tests
    └── fixtures/
        ├── sample_features.parquet    # Valid feature rows for inference tests
        ├── anomalous_records.parquet  # Labeled anomaly records for accuracy tests
        ├── invalid_features.parquet   # Malformed/out-of-range inputs for validation tests
        └── forecast_history.parquet   # Historical attendance series for forecast tests
```

## Running ML Tests

```bash
# All ML tests
pytest ml/tests/ -v

# Single model
pytest ml/tests/test_xgboost_model.py -v

# With detailed output on failure
pytest ml/tests/ -v --tb=long

# With coverage
pytest ml/tests/ --cov=ml --cov-report=term-missing
```

## XGBoost — Performance Predictor Tests

### Input Validation Tests

```python
# test_xgboost_model.py
import pytest
import pandas as pd
import numpy as np
from ml.models.performance_predictor import PerformancePredictor

@pytest.fixture
def model():
    return PerformancePredictor.load('gs://attendance-ml-artifacts/performance-predictor/xgb_v3.pkl')

@pytest.fixture
def valid_features():
    return pd.read_parquet('ml/tests/fixtures/sample_features.parquet')

class TestXGBoostInputValidation:

    def test_rejects_missing_required_feature(self, model, valid_features):
        """Model raises ValueError if a required feature column is missing."""
        incomplete = valid_features.drop(columns=['attendance_rate_30d'])
        with pytest.raises(ValueError, match="Missing required feature: attendance_rate_30d"):
            model.predict(incomplete)

    def test_rejects_null_values_in_features(self, model, valid_features):
        """Model raises ValueError if any feature contains NaN."""
        with_nulls = valid_features.copy()
        with_nulls.loc[0, 'avg_work_hours_30d'] = np.nan
        with pytest.raises(ValueError, match="NaN values detected in features"):
            model.predict(with_nulls)

    def test_rejects_out_of_range_attendance_rate(self, model, valid_features):
        """Attendance rate must be in [0, 100]."""
        out_of_range = valid_features.copy()
        out_of_range.loc[0, 'attendance_rate_30d'] = 150.0
        with pytest.raises(ValueError, match="attendance_rate_30d out of range"):
            model.predict(out_of_range)

    def test_accepts_minimum_valid_input(self, model):
        """Model accepts a single valid row without error."""
        single_row = pd.DataFrame([{
            'attendance_rate_30d': 90.0,
            'attendance_rate_90d': 88.0,
            'avg_work_hours_30d': 7.5,
            'late_login_count_30d': 2,
            'idle_ratio_30d': 0.05,
            'overtime_frequency_30d': 0.2,
            'attendance_consistency_score': 0.85,
            'login_time_deviation': 0.3,
            'attendance_streak': 15,
            'work_hour_variance_14d': 0.4
        }])
        result = model.predict(single_row)
        assert result is not None
```

### Output Range Tests

```python
class TestXGBoostOutputRange:

    def test_predictions_within_score_bounds(self, model, valid_features):
        """All predicted scores must be in [0, 100]."""
        predictions = model.predict(valid_features)
        assert predictions['score'].between(0, 100).all(), \
            f"Scores out of range: {predictions[~predictions['score'].between(0, 100)]}"

    def test_prediction_includes_confidence_bounds(self, model, valid_features):
        """Predictions must include lower and upper bound columns."""
        predictions = model.predict(valid_features)
        assert 'lower_bound' in predictions.columns
        assert 'upper_bound' in predictions.columns
        assert (predictions['lower_bound'] <= predictions['score']).all()
        assert (predictions['upper_bound'] >= predictions['score']).all()

    def test_30_day_forecast_length(self, model, valid_features):
        """Forecast must return exactly 30 predictions per employee."""
        result = model.predict_forecast(valid_features.iloc[[0]])
        assert len(result['predicted_scores']) == 30
```

### Accuracy Test

```python
class TestXGBoostAccuracy:

    def test_mae_within_threshold(self, model):
        """MAE on the held-out test set must be < 6.0."""
        test_data = pd.read_parquet('ml/tests/fixtures/sample_features.parquet')
        X_test = test_data.drop(columns=['actual_score'])
        y_test = test_data['actual_score']

        predictions = model.predict(X_test)['score']
        mae = (abs(predictions - y_test)).mean()

        assert mae < 6.0, f"MAE {mae:.2f} exceeds threshold of 6.0"
```

## Logistic Regression — Classifier Tests

```python
# test_logistic_classifier.py
from ml.models.employee_classifier import EmployeeClassifier

class TestClassifierInputValidation:

    def test_rejects_unknown_feature_column(self, classifier, valid_features):
        extra = valid_features.copy()
        extra['unknown_column'] = 0
        with pytest.raises(ValueError, match="Unexpected feature columns"):
            classifier.predict(extra)

    def test_accepts_all_valid_classes(self, classifier, valid_features):
        result = classifier.predict(valid_features)
        valid_classes = {'consistent', 'irregular', 'high_risk'}
        assert set(result['classification'].unique()).issubset(valid_classes)

class TestClassifierOutputRange:

    def test_probabilities_sum_to_one(self, classifier, valid_features):
        result = classifier.predict(valid_features)
        prob_sum = (
            result['prob_consistent'] +
            result['prob_irregular'] +
            result['prob_high_risk']
        )
        assert (prob_sum.round(4) == 1.0).all(), "Class probabilities do not sum to 1.0"

    def test_classification_matches_argmax_probability(self, classifier, valid_features):
        result = classifier.predict(valid_features)
        for _, row in result.iterrows():
            probs = {
                'consistent': row['prob_consistent'],
                'irregular':  row['prob_irregular'],
                'high_risk':  row['prob_high_risk']
            }
            expected_class = max(probs, key=probs.get)
            # Allow for high_risk_threshold override
            if row['prob_high_risk'] >= 0.35:
                expected_class = 'high_risk'
            assert row['classification'] == expected_class

class TestClassifierAccuracy:

    def test_weighted_f1_within_threshold(self, classifier):
        from sklearn.metrics import f1_score
        test_data = pd.read_parquet('ml/tests/fixtures/sample_features.parquet')
        X = test_data.drop(columns=['true_classification'])
        y_true = test_data['true_classification']
        y_pred = classifier.predict(X)['classification']
        f1 = f1_score(y_true, y_pred, average='weighted')
        assert f1 > 0.80, f"Weighted F1 {f1:.3f} below threshold of 0.80"
```

## Isolation Forest — Anomaly Detector Tests

```python
# test_isolation_forest.py
from ml.models.anomaly_detector import AnomalyDetector

class TestAnomalyDetectorInputValidation:

    def test_rejects_null_work_hours(self, detector, valid_records):
        with_null = valid_records.copy()
        with_null.loc[0, 'work_hours'] = np.nan
        with pytest.raises(ValueError):
            detector.score(with_null)

    def test_rejects_negative_confidence_score(self, detector, valid_records):
        invalid = valid_records.copy()
        invalid.loc[0, 'face_confidence'] = -0.5
        with pytest.raises(ValueError, match="face_confidence out of range"):
            detector.score(invalid)

class TestAnomalyDetectorOutputRange:

    def test_anomaly_scores_in_zero_one_range(self, detector, valid_records):
        result = detector.score(valid_records)
        assert result['anomaly_score'].between(0, 1).all()

    def test_is_anomaly_is_boolean(self, detector, valid_records):
        result = detector.score(valid_records)
        assert result['is_anomaly'].dtype == bool

class TestAnomalyDetectorAccuracy:

    def test_precision_on_labeled_anomalies(self, detector):
        """Precision on labeled anomaly test set must be > 0.72."""
        from sklearn.metrics import precision_score
        test_data = pd.read_parquet('ml/tests/fixtures/anomalous_records.parquet')
        X = test_data.drop(columns=['true_label'])
        y_true = test_data['true_label']  # 1 = anomaly, 0 = normal
        result = detector.score(X)
        y_pred = result['is_anomaly'].astype(int)
        precision = precision_score(y_true, y_pred)
        assert precision > 0.72, f"Precision {precision:.3f} below threshold 0.72"

    def test_known_simultaneous_location_flagged(self, detector):
        """A record with login_time_deviation > 5 sigma must always be flagged."""
        extreme_case = pd.DataFrame([{
            'work_hours': 8.0,
            'login_hour': 3.0,      # 3 AM — extreme deviation
            'login_time_deviation': 8.5,
            'attendance_streak': 0,
            'idle_ratio': 0.9,
            'gis_validated': 0,
            'face_confidence': 0.51
        }])
        result = detector.score(extreme_case)
        assert result.iloc[0]['is_anomaly'] == True
```

## Prophet — Attendance Forecaster Tests

```python
# test_prophet_forecast.py
from ml.models.attendance_forecaster import AttendanceForecaster

class TestForecastInputValidation:

    def test_rejects_series_shorter_than_minimum(self, forecaster):
        """Series must have at least 30 data points."""
        short_series = pd.DataFrame({
            'ds': pd.date_range('2026-03-01', periods=10, freq='B'),
            'y':  [20.0] * 10
        })
        with pytest.raises(ValueError, match="Insufficient history"):
            forecaster.fit_and_predict(short_series, horizon=7)

    def test_rejects_negative_headcount(self, forecaster, valid_series):
        invalid = valid_series.copy()
        invalid.loc[0, 'y'] = -5.0
        with pytest.raises(ValueError, match="Negative headcount"):
            forecaster.fit_and_predict(invalid, horizon=7)

class TestForecastOutputRange:

    def test_forecast_horizon_is_correct(self, forecaster, valid_series):
        result = forecaster.fit_and_predict(valid_series, horizon=7)
        future_dates = result[result['ds'] > valid_series['ds'].max()]
        assert len(future_dates) == 7

    def test_predicted_headcount_non_negative(self, forecaster, valid_series):
        result = forecaster.fit_and_predict(valid_series, horizon=7)
        assert (result['yhat'] >= 0).all()

    def test_lower_bound_less_than_upper_bound(self, forecaster, valid_series):
        result = forecaster.fit_and_predict(valid_series, horizon=7)
        assert (result['yhat_lower'] <= result['yhat_upper']).all()

class TestForecastAccuracy:

    def test_mape_within_threshold(self, forecaster):
        """MAPE on walk-forward validation must be < 10%."""
        from prophet.diagnostics import cross_validation, performance_metrics
        history = pd.read_parquet('ml/tests/fixtures/forecast_history.parquet')
        model = forecaster.get_fitted_model(history)
        df_cv = cross_validation(model, initial='90 days', period='14 days', horizon='7 days')
        df_perf = performance_metrics(df_cv)
        mape = df_perf['mape'].mean()
        assert mape < 0.10, f"MAPE {mape:.4f} exceeds threshold 0.10"
```

## Model Consistency Tests

All models must produce identical output for the same input across multiple calls (seed-controlled determinism):

```python
# test_model_consistency.py

class TestModelConsistency:
```

## SHAP Explainer Tests

```python
# test_shap_explainer.py

class TestSHAPExplainer:

    def test_shap_values_match_feature_count(self, explainer, valid_features):
        """Number of SHAP values must equal number of input features."""
        shap_values = explainer.explain(valid_features.iloc[[0]])
        assert len(shap_values[0]) == valid_features.shape[1]

    def test_shap_values_sum_to_prediction_delta(self, explainer, model, valid_features):
        """Sum of SHAP values must approximately equal prediction - base_value."""
        row = valid_features.iloc[[0]]
        prediction = model.predict(row)['score'].iloc[0]
        shap_vals, base_val = explainer.explain_with_base(row)
        shap_sum = base_val + sum(shap_vals[0])
        assert abs(shap_sum - prediction) < 0.5, \
            f"SHAP sum {shap_sum:.2f} does not match prediction {prediction:.2f}"

    def test_summary_string_is_non_empty(self, explainer, valid_features):
        summary = explainer.generate_summary(valid_features.iloc[[0]])
        assert isinstance(summary, str) and len(summary) > 10
```

## Interpreting Test Failures

| Failure Type | Likely Cause | Investigation Steps |
|---|---|---|
| **Input validation failure** | Schema change in pipeline output; new null-producing transformation | Check `data-pipeline-overview.md`, inspect latest ML-ready Parquet schema |
| **Output range violation** | Model predicting outside expected bounds — possible data leak or extreme input | Check training data distribution, look for outliers in recent feature data |
| **Accuracy below threshold** | Model drift — training data no longer represents current behavior | Trigger manual retraining, compare feature distributions between train and current data |
| **Determinism failure** | Random seed not fixed in model or scaler; non-deterministic preprocessing step | Audit model init params for `random_state`; check preprocessing for any stochastic step |
| **SHAP sum mismatch** | Model version mismatch between explainer and predictor | Ensure explainer and model are loaded from the same version artifact |
| **Fixture-based test failure** | Fixture data is stale or does not match current feature schema | Regenerate fixtures using `ml/tests/fixtures/regenerate_fixtures.py` |

### Regenerating Test Fixtures

```bash
python ml/tests/fixtures/regenerate_fixtures.py \
  --source=gs://attendance-pipeline-ml-ready/latest/ \
  --output=ml/tests/fixtures/
```

This pulls the latest ML-ready data, samples representative records, and writes fresh fixture files. Commit the updated fixtures alongside any schema changes.
